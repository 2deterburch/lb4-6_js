# lb4-6_js
Фільмова база
# Опис проєкту
Це односторінковий веб-застосунок, який отримує дані про фільми з TV Maze API та відображає їх у вигляді карток.
Користувач може шукати фільми за назвою та сортувати їх.

# Мета роботи
Закріпити знання з асинхронного програмування (Fetch API, async/await)
Навчитися працювати з API
Відпрацювати роботу з DOM
Реалізувати фільтрацію та сортування

# Функціонал
- Завантаження даних з API
- Відображення фільмів у вигляді карток
- Пошук за назвою
- Сортування:
-- за алфавітом
-- за рейтингом
- Обробка помилок

# Висновок
У ході роботи було створено веб-застосунок, який демонструє роботу з API, асинхронні запити, обробку даних та взаємодію з DOM. Було реалізовано фільтрацію та сортування, що робить застосунок інтерактивним і зручним для користувача.
